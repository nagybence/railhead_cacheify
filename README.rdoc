= RailheadCacheify

RailheadCacheify is a Ruby on Rails plugin that helps caching model functions.

== Installation

Installation is available as gem (recommended):

  config.gem 'railhead_cacheify'

Or as Rails plugin:

  $ ruby script/plugin install git://github.com/nagybence/railhead_cacheify.git
